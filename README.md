# ERP — Рівне Борошно

ERP-система управління виробничими процесами борошномельного підприємства.
Технології: Python 3.10+, PyQt5, Neon Database (PostgreSQL).

## Встановлення

### 1. Встановіть залежності
```bash
pip install -r requirements.txt
```

### 2. Налаштуйте підключення до Neon Database

Відкрийте файл `database/db.py` і замініть параметри підключення на ваші з neon.tech:

```python
DB_CONFIG = {
    "host":     "ep-XXXXX.eu-central-1.aws.neon.tech",  # ← ваш хост
    "database": "neondb",
    "user":     "neondb_owner",
    "password": "ваш_пароль",                            # ← ваш пароль
    "port":     5432,
    "sslmode":  "require",
}
```

Де знайти ці дані:
1. Зайдіть на https://neon.tech і відкрийте ваш проєкт
2. Перейдіть у розділ "Connection Details"
3. Скопіюйте рядок підключення — там є host, user, password, database

### 3. Ініціалізуйте базу даних (перший запуск)
```bash
python -c "from database.db import init_database; init_database()"
```

Це автоматично створить всі таблиці і додасть початкові демо-дані.

### 4. Запустіть програму
```bash
python main.py
```

## Функціонал

### Головна
- KPI-картки: замовлення, склад, виробництво, персонал
- Графік виробництва за 7 днів
- Прогрес виконання плану
- Таблиця останніх замовлень

### Замовлення
- Список замовлень з фільтром і пошуком
- Створення нового замовлення (форма)
- Редагування та скасування
- Автоматичний розрахунок суми

### Виробництво
- Поточна зміна (відкрити/закрити)
- Виробничий план на день
- Прогрес-бари виконання

### Склад
- Вкладки: Сировина / Готова продукція
- Кольорові індикатори рівня залишків

### Постачальники
- База постачальників з рейтингом
- Пошук і фільтрація

### Співробітники
- Список персоналу зі статусами
- Пошук за ПІБ або посадою

### Звіти
- 5 видів звітів
- Вибір періоду
- Попередній перегляд у таблиці

## Примітки

- Якщо БД недоступна — програма запускається в демо-режимі з тестовими даними
- Кольорова схема: помаранчевий акцент (#f5a623), темно-синя бокова панель
